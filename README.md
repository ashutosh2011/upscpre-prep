# UPSC Prelims 9-Day Drill

A static practice app for UPSC CSE Prelims GS Paper I elimination and guessing practice.

## Run locally

```bash
python3 -m http.server 8765
```

Then open:

```text
http://localhost:8765/index.html
```

## Deploy

This repo includes a GitHub Pages workflow. Push to `main` and the site will deploy through GitHub Actions.
